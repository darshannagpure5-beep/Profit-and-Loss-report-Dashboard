# Profit-and-Loss-report-Dashboard

## P&L Financial Dashboard | Power BI
An interactive Profit & Loss (P&L) dashboard built using Power BI to analyze revenue, costs, and profitability with dynamic insights.


## 🚀 Key Highlights

- 💰 Total Revenue analyzed: $118M+
- 📉 COGS Ratio: 72.2% (+0.8pp vs Previous Year)
- 📈 Revenue Growth: +9.3% YoY
- 💵 Net Profit increased by 6.5%
- 🌍 Top segment contributed ~38% of total revenue

---

## 📊 Dashboard Features

- Dynamic P&L Statement (Revenue, COGS, Gross Profit, Net Profit)
- Year-over-Year (YoY) and YTD analysis
- Waterfall chart for profit breakdown
- KPI Cards with growth indicators
- Interactive slicers (Year, Country, Product)
- Dynamic titles based on filters

---

## 🛠 Tools & Technologies

- Power BI  
- DAX (Data Analysis Expressions)  
- Data Modeling (Star Schema)  
- Excel (Financial Dataset)

---

## 🧠 Key Concepts Used

- Time Intelligence (YoY, YTD, SAMEPERIODLASTYEAR)
- Data Modeling using Dim_Date table
- Measures vs Calculated Columns
- DIVIDE function for safe calculations
- Dynamic reporting using SELECTEDVALUE

---

## 📁 Dataset

- Financial Sample Dataset (Microsoft)
- Contains:
  - Segment, Country, Product
  - Sales, COGS, Profit
  - Year, Month, Date
